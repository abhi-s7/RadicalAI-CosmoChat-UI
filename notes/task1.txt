

Install the necessary dependencies:

    - react-router-dom: used for routing
    - axios: for http requests
    - react-redux: for state management

After that, you need to install all the dependencies present in package.json: npm install package.json
This will create a package-lock.json 

npm install react-router-dom
npm install axios
npm install react-redux
npm install package.json

Rex recommends to install like: 
npm install react-router-dom axios react-redux @material-ui/core @material-ui/icons

also it says to do : npm install instead of npm install package.json

Option 1: Downgrade React
Downgrade react to a version that is compatible with @material-ui/core@4.12.4.
npm install react@17.0.2 react-dom@17.0.2

Option 2: Use --legacy-peer-deps
Force npm to install the dependencies by ignoring the peer dependency conflict.
npm install react-router-dom axios react-redux @material-ui/core @material-ui/icons --legacy-peer-deps

Option 3: Upgrade Material-UI
Upgrade @material-ui/core to the latest version (now known as @mui/material), which supports React 18.
npm install @mui/material @emotion/react @emotion/styled
npm install @mui/icons-material
